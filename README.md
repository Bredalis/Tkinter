<p>
  🖥️ Tkinter es la biblioteca estándar de Python para crear interfaces gráficas de 
  usuario (GUI). 🌟 Es conocida por su simplicidad y facilidad de uso, permitiendo a los 
  desarrolladores crear aplicaciones gráficas de manera rápida y eficiente. 🧩 Tkinter 
  proporciona un conjunto de herramientas y widgets básicos para construir interfaces 
  interactivas, siendo una excelente opción para proyectos tanto pequeños como grandes.
</p>
